---

This is the official ChonkyStation4 compatibility list.  
Please follow the following rules:  
- ### Use official releases of the emulator only. 
- ### Fill every field of the issue template (where applicable). 
- ### Do not create duplicate issues.
  - Different title IDs (releases) of the same game are OK.
  - Different OS is also OK.
  - The title ID is a string that looks like "CUSA02394".

---

This is the meaning of the different compatibility statuses:  
- **status-playable**: The game is playable without any major bugs or performance issues.
- **status-ingame**: The game works, but with major bugs or performance issues that make it unplayable.
- **status-menus**: The game reaches the main menus and crashes when trying to start the game.
- **status-boots**: The game displays some intro screens or plays audio, but does not reach the main menus.
- **status-nothing**: The game crashes or hangs before outputting any graphics or sound.
